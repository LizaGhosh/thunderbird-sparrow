Metric Explanations
Why These Metrics Matter:
  • Total Inputs: Total number of inputs processed for each category
  • Schema Validity: Percentage of outputs that comply with expected schema
  • Processing Success Rate: Overall success rate of processing operations
  • Accuracy: Field-by-field accuracy scores from CSV validation
Production Impact:
  • High Accuracy: Reduced manual intervention, faster processing
  • Low Accuracy: Increased errors, maintenance delays, safety risks
  • Schema Compliance: Ensures data quality and system reliability
  • Processing Success: Indicates system stability and error handling
