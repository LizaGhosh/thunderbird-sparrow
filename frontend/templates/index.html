System Overview

This system provides a comprehensive evaluation framework for industrial maintenance voice note parsing, featuring:

  • Work Item Triaging: Categorizes maintenance requests and tasks
  • Closing Comments: Generates completion reports with downtime extraction
  • Multi-Model Support: Configurable AI providers (Claude, Gemini)
  • Pydantic Validation: Robust schema validation ensuring data quality
  • Performance Metrics: Detailed accuracy analysis across all criteria
  • Voice Processing: Audio recording and AI transcription
